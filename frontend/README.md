# Notisight Frontend

Vite/React frontend for the Notisight application.

## Local Development

1. Install dependencies:
   `npm install`
2. Optional local API override:
   `VITE_API_URL=http://localhost:5228`
3. Run the development server:
   `npm run dev`

## Production Environment

Set the public API URL in the hosting provider:

```env
VITE_API_URL=https://api.notisight.dev
```
